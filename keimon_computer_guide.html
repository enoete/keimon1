📡 ICT, Computers & Types

🏫 What Even IS a Computer?

Think about your classroom, Teacher Keimon. Little Shiloh walks in with some finger paintings (that's INPUT — raw data coming in). You look at them, understand them, and give Shiloh feedback (that's PROCESSING). Then you display Shiloh's work on the wall for everyone to see (that's OUTPUT).

A computer works the exact same way! Data goes IN → the computer PROCESSES it → Information comes OUT. You are basically a biological computer, Teacher Keimon! 😄

🖥️ Official Definition: A computer is an electronic digital device that stores, retrieves, and processes data — and can be programmed with instructions. It has hardware (the physical parts you can touch) and software (the invisible instructions that tell it what to do).

🌐 ICT vs IT — What's the Difference?

Imagine Makaila and Makaili playing together. They look similar but are different kids doing different things!

Term What It Means Real Life Example
IT (Information Technology) Computers + hardware + software to collect, process and store info The computer lab at school
ICT (Info & Communications Technology) IT + internet + phones + TV + radio — everything that communicates WhatsApp + YouTube + the lab computer all working together

Oma Lu put it best: "IT is the tools. ICT is the tools PLUS the conversations you have with those tools!" 😄

⌨️ Computer Peripheral Devices

Think of the computer as little Aviana sitting at her desk. Aviana alone can't do much — she needs TOOLS! Her pencil (keyboard), her eyes on the board (monitor), her hand raising for attention (mouse)... those are all peripherals!

📥 INPUT Devices (things that send info TO the computer): Keyboard, Mouse, Scanner, Microphone, Camera

📤 OUTPUT Devices (things the computer sends info OUT to): Monitor/Screen, Printer, Speakers

💾 Storage Devices: Hard Drive, USB drive, CD/DVD — where info is saved!
Storage sizes go: Bit → Byte (8 bits) → Kilobyte (KB) → Megabyte (MB) → Gigabyte (GB) → Terabyte (TB). Think of a Byte like one letter. A Terabyte is like storing every book in a giant library! 📚
⏳ Computer Generations

👶→👦→🧑→👨→🧙 Growing Up!

Teacher Keimon, you've watched children grow from babies to big kids. Well, computers grew up too — in 5 amazing "generations"! Each generation got smaller, faster, cheaper, and smarter. Just like Geron went from needing help tying his shoes to being completely independent!

🏠

1st Generation (1946–1958) — The Giant Baby!

Used vacuum tubes — think big glass light bulbs. These computers weighed 30 TONS and filled 15,000 sq. feet — that's like your entire school! They generated SO much heat they needed their own air conditioning. Imagine a baby that needs an entire building just to exist! 🍼

Processing speed: milliseconds (thousandths of a second). Stored data on punched cards!
📦

2nd Generation (1959–1964) — Toddler Stage

Replaced vacuum tubes with transistors. One transistor could do the work of 40 vacuum tubes! Computers got smaller, cheaper, faster, and used less power. Like when Lea went from crawling to walking — suddenly so much more capable! 🧒

Processing speed: microseconds (millionths of a second). Used assembly language!
🔬

3rd Generation (1964–1970) — School Age!

Tiny Integrated Circuits (ICs) put multiple transistors on one tiny chip! Computers could run multiple programs at once. Operating Systems began here. Like when Blake started kindergarten and suddenly could multitask — eat, draw, AND listen to a story! 🎨

Processing speed: nanoseconds (billionths of a second). Minicomputers emerged!
🏆

4th Generation (1971–1984) — Teenager!

Microprocessors put an ENTIRE computer on ONE chip! The personal computer (PC) was born. Laptops were invented in 1982. C++, Java programming languages appeared. Like Janae getting her first phone — suddenly independent and doing everything herself! 📱

This is when Windows and Mac OS were born. The PERSONAL computer era began!
🤖

5th Generation (1985–Now) — Adult & Beyond!

Artificial Intelligence, robots, parallel processing! Computers can now "learn" and "think" — like an AI tutor for your students! Oma Lu is amazed. Robots don't eat, don't sleep, don't strike, don't need wages... but they still can't replace the love of a teacher like you! ❤️

We are HERE right now. AI like ChatGPT, robots in factories, self-driving cars — all 5th generation!

🖥️ The Operating System

👩‍🏫 The OS is YOU, Teacher Keimon!

Seriously! The Operating System (OS) is the software that acts as the intermediary between the user and the hardware. In your classroom:

🎒 The students = Application programs (Word, Chrome, games)
🏫 The building & furniture = Hardware (computer, keyboard, monitor)
👩‍🏫 YOU (the teacher) = the Operating System!

You manage the classroom, give students access to resources, make sure nobody fights over the crayons, keep things running smoothly... That's EXACTLY what an OS does for a computer!

🎯 OS Goals:
1. Execute user programs (run apps) and make solving problems easier
2. Make the computer convenient and friendly to use
3. Use the computer hardware efficiently

🏗️ The Layered Structure — Floors in a School

Computer resources are organized in layers, where each layer only interacts with the layer directly above or below it. Think of your school as a 4-story building:

👤 USER (You — 4th Floor)
📱 APPLICATIONS (Apps — 3rd Floor)
🖥️ OPERATING SYSTEM (2nd Floor)
🔧 HARDWARE (Ground Floor)

You (User) don't go directly to the basement to turn on the lights. You flip a switch (Application) which tells building management (OS) which then controls the electrical system (Hardware)!

Key Rule: Install new hardware? Only the OS needs updating (not your apps). Install new software? Only the OS needs to support it (not the hardware). Change the OS? BOTH apps AND hardware must be compatible!

🎛️ The 6 Functions of an OS — Meet the Teaching Staff!

🏫
Oma Lu narrates...

"Honey, running a school takes a whole STAFF, not just one person! The Operating System is the same — it has 6 big jobs, and each one is like a different teacher or staff member!"

OS Function School Analogy
1. 🔄 Process Management The Schedule Coordinator — decides which class (program) runs when and for how long
2. 🧠 Memory Management The Resource Room Teacher — tracks who has what supplies and gives them out fairly
3. 📁 File Management The Librarian — organizes, creates, finds, and backs up all documents
4. 🖨️ Device (I/O) Management The IT Tech — manages printers, keyboards, mice; decides who gets to use them
5. 🔒 Security & Protection The Security Guard — passwords, permissions, no unauthorized access!
6. 🖥️ User Interface The School Receptionist — the friendly face through which you interact with everything

💾 Memory Management Deep Dive

Think of computer memory like your classroom's activity tables:

🪑 RAM (Random Access Memory) = The actual table space where kids work RIGHT NOW. When class ends (computer turns off), the table is cleared. It's volatile — everything disappears!

📚 Hard Drive Storage = The bookshelves and cabinets. Things stay there even when school is closed.

🪄 Virtual Memory = When the tables are full, you use a corner of the floor as overflow space. The OS uses part of the hard drive as "extra table space" — that's virtual memory!

📜 History of Operating Systems — The OS Grows Up!

Generation What Happened
1940s–early 1950s NO operating system! Programmers wired plug boards directly. Like teaching with no lesson plan at all!
1955–1965 (2nd Gen) First OS created — called GMOS by General Motors for IBM. Batch processing: one job at a time.
1965–1980 (3rd Gen) MULTIPROGRAMMING begins! Multiple programs running at once. Like Teacher Keimon teaching reading AND having a group doing crafts simultaneously!
1980–Present (4th Gen) Windows & Mac OS born! Personal computers for everyone. The friendly, click-able interface we use today!
🌍 Popular OS Categories Today:
🖥️ Desktop: Windows, macOS, Linux/Ubuntu
🖧 Server: Windows Server, Linux
📱 Mobile: Android, iOS
💡 CLI vs GUI — Two Ways to Talk to Your Computer:

CLI (Command Line Interface) = Like sending written notes to a student. You TYPE commands. Advanced, fast, but no pictures. (Used by tech experts!)

GUI (Graphical User Interface) = Like pointing at things on a bulletin board. You CLICK icons and menus. Friendly, visual, easy! (Windows, Mac — what you use!)
🪟 Navigating Windows 10

🏠 Your Digital Classroom — The Desktop

When you turn on a Windows 10 computer, you arrive at the Desktop — think of it like your physical classroom desk! It has everything you need to start working. Let's tour it together, just like you'd orient a new student on the first day of school! 🎒

⭐ The Start Menu — Your Master Key

🧒
Blake explains...

"Teacher Keimon, the Start Menu is like the school's office! If you need ANYTHING, you go there first. Click the Windows logo (⊞) at the bottom-left, or press the Windows key on the keyboard. From there you can find ANY program, shut down, or change settings!"

⊞ How to open: Click the Start button (bottom-left ⊞) OR press the Windows key on keyboard

Left side: Recently used apps + all installed programs
Right side (Tiles): Pinned apps you use most — like your favorite supplies on your desk!

🔌 Shut down/Restart: Start → Power button → choose Sleep, Shut down, or Restart

🔍 Search — Your Teaching Assistant!

Windows 10's search bar is like having Aeva as your personal assistant — she knows where EVERYTHING is! Just start typing in the search box (next to the Start button) and it finds:

📱 Apps — type "Word" to open Microsoft Word
📁 Files — type part of a filename to find it
🌐 Web — search the internet directly
🔊 Cortana — Microsoft's voice assistant (like Siri for Windows!)

🗂️ Task View & Virtual Desktops

👵
Oma Lu marvels...

"Lord have mercy, they call it 'Task View'? Back in my day you had ONE desk and ONE pile of papers! Now you press Windows + TAB and ALL your open programs spread out nice and neat so you can find exactly what you need. And Virtual Desktops? It's like having THREE desks in one room — one for work, one for personal, one for school!"

📋 Task View: Click the Task View button in taskbar OR press Windows + TAB
💻 Virtual Desktops: Create separate workspaces — one for class materials, one for personal browsing!
🪟 Snap Enhancement: Drag a window to the edge of the screen to snap it into place — work with TWO programs side by side!

⚙️ Settings — Personalizing Your Space

Just like you arrange YOUR classroom exactly how YOU like it, Windows 10 lets you personalize everything! Go to Start → Settings (⚙️):

Setting What You Can Do School Equivalent
🖥️ System → Display Change screen resolution & text size Adjusting the whiteboard height for your students
🖨️ Devices Add printers, mice, keyboards Connecting the classroom printer
🎨 Personalization Wallpaper, colors, lock screen, themes Decorating your classroom walls!
👤 Accounts User accounts & passwords Student name tags & class lists
Can't see things well? Go Settings → System → Display → "Scale and Layout" → increase the percentage to make EVERYTHING bigger on screen. No more squinting! 👓

📂 File Explorer — The School Filing Cabinet

File Explorer is your digital filing cabinet! Right-click the Start button → Select "File Explorer" (or press Windows + E). Then click "This PC" to see all your drives:

💾 C: Drive (Local Disk C) = Your main filing cabinet — where Windows and programs live
📁 Documents, Pictures, Downloads = Pre-made folders like labeled cabinet drawers
🔌 USB Drive (shows as D: or E:) = A portable folder you carry with you, like a folder in your bag!

To add a desktop shortcut: Click Start → Find the app → Hold left mouse button → Drag to the desktop. Like taping a sticky note with your most-used tools right on your desk! 📌

🚪 Logging Out & Switching Users

To Sign Out: Start Menu → Click your User Icon at the top → Select "Sign out"

To Switch Users: Same menu → "Switch account" (your session stays open in the background!)

Quick Lock Screen: Press Windows + L — locks your screen without signing out. Great when you step away briefly!
